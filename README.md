# Real Estate Price Analysis Dashboard

This project presents an exploratory data analysis and evaluation of a Linear Regression model on the California Housing dataset.

## Technologies Used
- Python
- NumPy, Pandas, Matplotlib
- Scikit-learn
- HTML, CSS

## Features
- Dataset overview
- Housing price distribution
- Feature correlation analysis
- Model performance metrics

## How to Run
1. Run `ml_code.py` to generate plots
2. Open `index.html` in a browser
